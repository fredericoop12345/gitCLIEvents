This project was done on: https://roadmap.sh/projects/github-user-activity
This project practises building a simple command-line interface for to fetch the recent activities of a github user.
The following is the command to type to run it:
-github-activity <username>

Requirement:
Have the latest version of python installed.

Type this in the terminal to open the .bash_profile
    vi ~/.bash_profile

Press i to enter normal mode

Type the following on a line:
    alias github-activity="python3 /Users/fredrickoppong/gitCLIEvents/githubEvents.py"

Type ':' to the screen followed with wq and press the Enter key.

Type 'source ~/.bash_profile in the terminal before running the CLI Interface
